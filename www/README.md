##Transit Alert
######(working title)

Real-time bus arrival times for NJ Transit passengers. Select a bus line, travel direction, and stop name for upcoming arrivals.

######Features in development:
$ Set alerts to get notified when buses are in your area
$ Save favorite routes and stops for quick reference


###### To run (requires Ionic and Angular):

$ ionic platform add ios
$ ionic build ios
$ ionic emulate ios

Substitute ios for android if not on a Mac, but if you can, the ios development toolchain is a lot easier to work with until you need to do anything custom to Android. Or, use "ionic serve" to launch in a browser.
